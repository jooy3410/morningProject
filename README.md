# 미라클모닝

개인프로젝트
<br><br>
기간 : 2021.05.17 ~ 2021.07.23
<br><br>
이른 아침에 일어나 독서, 운동 등 자기개발을 하는 사람들을 위해 아침루틴을 관리하고 기록할 수 있는 서비스 입니다.
<br><br>

* 적용기술 
  * Back-End  
    * Spring Framework 
    * JAVA 
  * DevOps 
    * MariaDB
    * MongoDB
    * AWS EC2
    * RDS
  * Front-End 
    * HTML 
    * CSS 
    * Javascript 
    * BootStrap 
  * 기타 
    * Open API
